
### 注册

**请求URL：**
- /user
**请求方式：**
- POST

**参数：**

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|account |是  | 账号   |
|pwd |是  | 密码   |
|role |是  | 0为普通用户，1为管理员用户|

**返回示例**

 ```
 {
    "code": 1,
    "msg": "success",
 }
 {
    "code": 0,
    "msg": "fail",
 }
 ```

### 场馆删除

**简要描述：**
- 场馆删除接口

**请求URL：**
- /gymDetail

**请求方式：**
- DELETE

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|gymId |是  |Number |查询条件(场馆id)   |

 **返回示例**

 ```js
 {
    "code": 1,
    "msg": "success",
}
 ```

  **返回参数说明** 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|ref_area |number   |对应校区id  |
|status |number   |是否对外开放 0:开放 1:维修不对外开放  |
